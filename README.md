# claw machine game

a simple HTML + CSS + vanilla JS claw machine game

![claw machine by studi0.rih](https://github.com/user-attachments/assets/9b00ef82-3088-4652-82a2-9f3604f22e42)

## how to play

* start by pressing the right button, lift to release
* then move the claw up
* try to align the shadow with a plushie
* good luck!
